# Stronghold® Branding Kit

![Stronghold®](Stronghold%20Logos%20-%20March%202022/Blue/Stronghold%20Logo%20Type%20Blue.svg)

Public embed and reference hub for Stronghold® brand assets. Use these hosted logos in any markdown file, HTML page, email, or document — no local files needed.

---

## Quick Start

Drop the Stronghold® logo into any markdown file:

```markdown
![Stronghold®](https://raw.githubusercontent.com/strongholdpay/Stronghold-Branding-Kit-PUBLIC-FOR-ALL/main/Stronghold%20Logos%20-%20March%202022/Blue/Stronghold%20Logo%20Type%20Blue.svg)
```

Or in HTML:

```html
<img src="https://raw.githubusercontent.com/strongholdpay/Stronghold-Branding-Kit-PUBLIC-FOR-ALL/main/Stronghold%20Logos%20-%20March%202022/Blue/Stronghold%20Logo%20Type%20Blue.svg" alt="Stronghold®" height="40">
```

## Which Logo to Use

| Context | Variant |
|---------|---------|
| Light/white backgrounds | Blue Mark + Black Type combo (preferred), or Blue wordmark |
| Dark backgrounds | Blue Mark + White Type combo, or White-White combo |
| Full brand presence | Logo Combo lockups (logomark + wordmark together) |
| Wordmark only | Logotype — Blue (light BG) or White (dark BG) |
| Icon only (favicons, app icons) | Logomark — Blue Gradient or White |
| Partner/co-branded materials | "Powered by Stronghold" |
| SHx product branding | SHx Logo variants |
| Document headers & footers | Logo Combo at reduced size |
